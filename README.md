## Build Setup

```bash
# install dependencies
$ npm install
$ pip install -r requirements.txt

# serve with hot reload at localhost:5000
$ npm run dev

# run python socket server at localhost:8080
$ python main.py

# build for production and launch server
$ npm run build
$ npm run start

$ python main.py

```
